# 12. Integer to Roman
## 算法
先生成十进制数和罗马数的对应表：
```
romans = ["M","CM","D","CD","C","XC","L","XL","X","IX","V","IV","I"]
values = [1000, 900, 500, 400, 100, 90, 50, 40, 10, 9, 5, 4, 1]
```

拿到给定的数字后，就去遍历`values`，如果给定的数字比`values`当前的元素大，就减去当前元素。一直这样循环直到变为0。

## 复杂度
- 时间复杂度：`O(n)`，对应表的长度是固定的，主要看给定的数字大小；
- 空间复杂度：`O(1)`
